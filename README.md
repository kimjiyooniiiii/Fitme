## ✔ 개발 목표
카테고리 별로 상품을 찾고, 장바구니에 담아 모바일 결제까지 가능한 여성 웹 쇼핑몰. <br>
비회원 사용자에게도 장바구니 기능을 제공하고, 상품 상세 페이지에서도 미리 가격 계산을 할 수 있는 기능을 제공한다.

<br>

### 👤 개발 인원 (1명) : 김지윤 (Full-Stack)
